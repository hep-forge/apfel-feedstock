[![Anaconda Platforms](https://anaconda.org/hep-forge/apfel/badges/platforms.svg)](https://anaconda.org/hep-forge/apfel)

Feedstock for [apfel](http://apfel.mi.infn.it/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

APFEL: A PDF Evolution Library

## Install

```bash
conda install -c hep-forge -c conda-forge apfel
```
